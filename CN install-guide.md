# 🦀 CrabClaw (螃蟹) 跨平台小白一键本地部署指南

不用敲代码，不用懂编程，只需会“双击”（或简单执行一句命令），即可完成属于你自己的全自主智能体（Local Agent）本地部署！

> ⚠️ **注意**：本版本为跨平台商业进化版，支持 Windows / macOS / Linux，且完全不需要你手动修改任何代码文件！

## 🎯 准备工作：所需文件

请在你的电脑桌面上新建一个普通的文件夹，命名为 `我的智能体`。
在这个文件夹里，请确保放置了以下文件（必须已完成“另存为”操作去掉 .txt 后缀）：

- `crabclaw.py` （核心大脑代码）
- `run_crabclaw.bat` （Windows 用户的专属一键启动器）
- `run_crabclaw.sh` （Mac / Linux 用户的专属启动器）

---

## 🚀 第一步：安装系统环境 (Python)

CrabClaw 的内核是 Python 写的，所以电脑里需要有这个环境。

### 🪟 对于 Windows 用户：傻瓜式自动检测

怎么知道我有没有装？ 直接双击 `run_crabclaw.bat`。如果弹出红字提示“未检测到 Python”，并自动为你打开了网页，那就是没装。

1. 在弹出的网页点击黄色的 **Download Python** 按钮。
2. 下载后双击安装包。
3. 🚨 **极其重要的一步（不勾绝对会失败）**：
   在安装界面的 **最底部**，有一个方框写着 `Add python.exe to PATH`（或 `Add Python to environment variables`），**一定要把它打上勾！** 然后点击 *Install Now* 即可。

### 🍎 对于 macOS 用户：

打开你电脑上的 **终端 (Terminal)**，输入以下命令并回车（推荐使用 Homebrew）：
```bash
brew install python
```
*(如果你没有 brew，可以直接去官网 python.org 下载 Mac 版安装包，一路“下一步”即可)*

### 🐧 对于 Linux 用户：

打开终端，根据你的系统类型输入：

**Ubuntu/Debian:**
```bash
sudo apt update && sudo apt install python3 python3-pip
```
**CentOS/RHEL:**
```bash
sudo yum install python3 python3-pip
```

## 🚀 第二步：一键点火，智能配置！

忘掉旧版本那种“用记事本打开代码修改”的繁琐步骤吧，系统现在会引导你！

### 👉 Windows 用户：
直接双击运行 `run_crabclaw.bat`。

### 👉 macOS / Linux 用户：
在终端中进入你的文件夹（例如：`cd ~/Desktop/我的智能体`），然后输入以下两行命令：
```bash
chmod +x run_crabclaw.sh
./run_crabclaw.sh
```

### 🤖 首次运行向导（所有系统通用）：

当你第一次成功启动时，黑色的命令行窗口会 **自动** 帮你下载缺少的运行组件（如 openai 等，带进度条）。下载完成后，系统会弹出友好的 **中文向导**，向你提问：

- **请输入您的 API Key**：把你在各大模型平台（如 DeepSeek、阿里云百炼）申请到的 `sk-xxxxxx` 密钥复制粘贴进去，按回车。
- **请输入 API Base URL**：如果你用的是 DeepSeek，直接按回车跳过即可（默认已填好）。
- **请输入您想调用的模型名称**：直接按回车跳过即可（默认 deepseek-chat）。

> ✅ **大功告成！** 你的配置会被自动保存在同目录的 `config.json` 里。以后双击启动，就会直接进入聊天界面：
> **“🦀 CrabClaw 商业级智能体已启动 🚀”**

## 🏥 常见问题与急救指南 (FAQ)

### ❓ Q1: (Windows) 安装完 Python 后，双击 bat 还是提示“未检测到 Python”？
* **解答**：百分之百是因为你安装时 **忘记勾选** 最底部的 `Add python.exe to PATH` 了！
* **救法**：打开控制面板卸载刚才装的 Python，重新双击安装包，**睁大眼睛把底部的勾打上**，重新安装一遍即可。

### ❓ Q2: 提示 [警报] 节点鉴权失败或已欠费停机？
* **解答**：你的大脑（大模型）出问题了，通常是 Key 填错或欠费。
* **救法**：
  1. 检查你填入的 API Key 是否少复制了一个字母，或者带有空格。
  2. 登录你申请 API Key 的平台（如 DeepSeek 官网），看看是不是账号里没钱了。充值 10 块钱能用非常久！
  *(想修改填错的 Key，只需用记事本打开同一目录下的 `config.json` 文件，修改后保存重启即可)*

### ❓ Q3: 提示缺少依赖包，或者界面卡在“正在自动为您下载必要组件”很长时间不动？
* **解答**：网络波动导致无法连接到清华大学的 Python 下载镜像。
* **救法**：关闭窗口重新双击运行一次。如果连续多次失败，您可以尝试对着 AI 智能体直接输入：*“我缺少了 xxx 包，请用你的 shell 工具帮我执行 pip install xxx 安装它”*，AI 会自动写代码帮你装好！

### ❓ Q4: (Mac/Linux) 提示 `Permission denied`（权限被拒绝）？
* **解答**：你的系统安全机制拦截了脚本运行。
* **救法**：打开终端，输入 `chmod +x run_crabclaw.sh` 赋予它可执行权限，然后再运行 `./run_crabclaw.sh` 即可。

### ❓ Q5: 为什么中文显示是乱码？
* **解答**：新版本代码中已经加入了强制 UTF-8 终极防乱码补丁。
* **救法**：如果还是有乱码，请尝试 **右键点击命令行窗口的最上方标题栏 -> 属性 -> 字体 -> 改为 新宋体 或 Consolas** 即可。
